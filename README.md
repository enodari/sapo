# sapo

Minimalistic Python SOAP server

---

### Highlights
* SOAP service generation from WSDL file.
* External WSDL/schemas support (only local paths).
* XML messages parsing, validating and building with lxml.

### Status
WIP, any help is welcome!

### Requirements
* Python 3.4
* lxml>=3.6.1
